Credit Card Fraud Detection is a machine learning based security system designed to identify suspicious transactions and reduce financial risks.

The system analyzes transaction patterns, applies advanced classification algorithms, handles class imbalance using SMOTE, and provides transparent predictions using Explainable AI.
